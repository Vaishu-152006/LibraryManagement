Library Management System

Project Description

The Library Management System is a simple console-based Java application designed to manage basic library operations. The system allows users to add books, register members, borrow and generate reports. This application operates completely in memory and does not use any external database or file system for storing data.

The main objective of this project is to demonstrate object-oriented programming concepts such as classes, objects, encapsulation, collections, and package structuring in Java. The system is menu-driven and interacts with the user through the console.
